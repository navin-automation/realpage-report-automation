# RealPage Report Automation

An automated tool built with Python to streamline property management reporting.

## Features
- Multi-browser support (Playwright & Selenium).
- Automated login and navigation for RealPage dashboards.
- Dynamic report downloading and renaming.

## Installation
1. Clone the repo: `git clone <your-repo-link>`
2. Install dependencies: `pip install -r requirements.txt`
3. Install Playwright browsers: `playwright install`
4. Add your credentials in `RRR.xlsx`.
